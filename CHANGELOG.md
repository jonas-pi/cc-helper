# CC 命令助手 - 更新日志

## v2.0.0 (2026-01-07)

### 新增功能
- ✨ **新增 `cc list` 命令**：专门列出所有可用模型
  - Ollama 模式：显示已安装的本地模型列表
  - 云端 API 模式：显示当前 API 配置信息
  - 高亮显示当前使用的模型

### 优化改进
- 🎨 **简化 `cc hello` 输出**：移除模型列表，更简洁
- 🔧 专注性更好：信息展示更清晰，功能更专一
- 🔧 `cc hello` 添加提示：引导用户使用 `cc list` 查看模型

### 重大变化
- ⚠️ `cc hello` 不再显示模型列表（使用 `cc list` 代替）

## v1.9.0 (2026-01-07)

### 新增功能
- 🪟 **Windows Shell 智能切换**：根据目标 Shell 生成不同语法的命令
  - PowerShell 模式：生成 PowerShell 命令（Get-Date, Get-Process等）
  - CMD 模式：生成 CMD 命令（echo %date%, tasklist等）
- ✨ 新增 `cc -shell` 命令：快速切换目标 Shell
- ✨ 自动检测父进程：在 CMD 中运行时自动使用 CMD 语法（实验性）
- ✨ `cc hello` 显示当前目标 Shell

### 优化改进
- 🔧 Windows 用户体验大幅提升，命令语法更准确
- 🔧 避免 PowerShell 命令在 CMD 中无法运行的问题
- 🔧 配置文件持久化目标 Shell 设置

## v1.8.0 (2026-01-07)

### 新增功能
- 🎯 **灵活安装选项**：首次安装可选本地或云端模式
  - 选项 1：本地安装（Ollama + 本地模型，免费离线）
  - 选项 2：云端 API（DeepSeek/豆包/通义千问等，无需下载模型）
- ✨ 云端模式跳过 Ollama 下载，直接配置 API
- ✨ 节省时间和磁盘空间，更适合只用云端 API 的用户

### 优化改进
- 🔧 安装体验更友好，用户可根据需求选择
- 🔧 云端模式安装后直接提示 `cc -config`
- 🔧 降低首次使用门槛
