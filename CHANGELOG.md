# CC 命令助手 - 更新日志

## v1.6.0 (2026-01-07)

### 新增功能
- 🌱 **API 接入系统**：cc -config 配置不同的 AI API
- ✨ 支持 OpenAI（GPT 系列）
- ✨ 支持 Anthropic（Claude 系列）
- ✨ 支持自定义兼容 OpenAI API 的服务（Groq, Together AI 等）
- ✨ 配置文件持久化（~/.cc_config）

### 优化改进
- 🔧 让 cc 有更多成长空间，不再局限于本地模型
- 🔧 统一的 API 调用接口，支持动态 API Key 认证
- 🔧 更新 README，添加 API 配置详细说明

## v1.5.0 (2026-01-07)

### 新增功能
- ✨ 添加版本管理系统，更新时显示版本对比和更新内容
- ✨ 自动编码检测，GBK/UTF-8 环境自动适配
- ✨ 工作模式（-w）和休息模式（-r）切换
- ✨ 增强的 cc hello 命令，显示所有已安装模型
- ✨ 模型管理命令：cc -change (切换), cc -add (安装), cc -del (删除)
- ✨ 增强的 cc -help 命令，美观结构化的帮助信息

### 优化改进
- 🔧 智能模型检测和自动切换，确保脚本运行健壮性
- 🔧 修复编码问题，中文显示更稳定
- 🔧 优化安装脚本，根据系统配置推荐模型，并支持多选
- 🔧 修复 cc -u 预设命令不依赖模型的问题
- 🔧 修复 install.sh 在 curl | bash 管道中无法读取用户输入的问题
- 🔧 修复 install.sh 语法错误 (缺少 fi)
- 🔧 修复 cc -u 更新 URL 错误
- 🔧 优化 cc hello 输出，更可爱和详细
- 🔧 优化模型信息，确保配置要求和特点准确
- 🔧 优化 Windows 模型推荐策略，优先 PowerShell 友好模型

### Bug 修复
- 🐛 修复 cc hello 在 GBK 编码下表情乱码问题
- 🐛 修复 cc -u 在模型不存在时报错的问题
